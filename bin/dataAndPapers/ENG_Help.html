Geometry Functions

Geometry Function panel contains what you would use to do math with geometry shapes, mostly stuff that is not on most calculator or computers. The table in the following is a detail listing of all functions.

Feature Name Feature Function Formula Source
Get Area of a Circle this is a tool that calculates area of a circle from diameter. A = π*r^2 how it is done
Get Perimeter of a Circle this is a tool that calculates perimeter of a circle from its diameter. C = 2*π*r how it is done
Get Rectangle Area this is a tool that calculates area of a rectangle, only rectangle, not any tetragon. A = bh how it is done
Get Sphere Volume this is a tool that calculates Sphere volume. V = 4/3πr^3 how it is done
Get Hypotenuse of Right Triangle this is a tool that calculates Hypotenuse of a Right Triangle. c = √(a^2 + b^2) how it is done
Get the Volume of Rectangular Prism this is a tool that calculates the Volume of a Rectangular Prism, only Rectangular Prism. V = Bh how it is done
Get Cylinder Volume this is a tool that calculates the Volume of a Cylinder. V = πr^2h how it is done
Get Cone Volume this is a tool that calculates the Volume of a Cone. V = 1/3(πr^2)h how it is done
